# Personal Portfolio Website

A modern, responsive portfolio website built with React and .NET.

## Project Structure

- `/frontend` - React TypeScript application with Tailwind CSS
- `/backend` - .NET API backend

## Technologies Used

### Frontend
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- React Router

### Backend
- ASP.NET Core
- Entity Framework Core
